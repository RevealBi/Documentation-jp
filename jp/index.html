Reveal について学びましょう

ユーザー向けドキュメント
ダッシュボードをすばやく作成、編集したり、エンタープライズ データソースを簡単にクエリ、フィルタしたりして、意味のあるデータ可視化を実現しましょう。Reveal ユーザー以外の人とも簡単に共有することができます。

開発者向けドキュメント
追加の要件なしで、データ アナリティクスをアプリケーションに簡単に統合できます。Reveal の柔軟なアーキテクチャと豊富な API によってアプリでの機能をコントロールすることで、アプリの一部としてのシームレスなエクスペリエンスをユーザーに提供します。

トレーニング ビデオ
サンプル ビジュアライゼーションとサンプル データを使用し、チュートリアルビデオに沿って Reveal を始めましょう。
